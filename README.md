# Godot_Snake
 A Classic Snake clone made with Godot 4.

## Implemented
- Snake Head & Tail Movements
- Grid map
- Infinite boundaries
- Rat spawn
- Grid blocks on background
- Cool rainbow Background
- Cool Head, Tail, Rat Texture

## KEYs
- **ESC** or **SPACE** (Pause: Restart Game)
- **W** or **TOP_ARROW** (Snake Top Movement)
- **A** or **LEFT_ARROW** (Snake Left Movement)
- **S** or **BOTTOM_ARROW** (Snake Bottom Movement)
- **D** or **RIGHT_ARROW** (Snake Right Movement)
